adsensehost v4.1

AdSense Host API

Generates performance reports, generates ad codes, and provides publisher management capabilities for AdSense Hosts.

Version:
  • v4.1

Example

var google = require('googleapis');
var adsensehost = google.adsensehost('v4.1');
